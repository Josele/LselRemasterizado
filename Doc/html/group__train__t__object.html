void trains_setup ( void  )

Model trains and DCC instantiation.

Sets up the DCC object (in charge of sending commands to the railway amplifier) and every train present in the model, attaching them to the DCC.

It also registers all of the train commands in the intepreter.

At the end, sets the current_train to the first one stored in the global trains array.
