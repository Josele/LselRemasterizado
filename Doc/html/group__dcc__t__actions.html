Add data packet to DCC buffer.

Formats a generic basic DCC packet with 7 bit address and 8 bit data, filling the preamble and separator bits and calculating the required CRC. Then adds it to the packet queue for transmission.

Parameters
thisthe dcc_sender that will transmit the packet

Add packet to DCC buffer.

Adds a packet to the DCC packet queue. Packets are serviced in a FIFO fashion. If the queue is full old packets will be discarded to make room for new ones.
