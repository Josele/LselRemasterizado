Model trains and D\-C\-C instantiation. 

Sets up the D\-C\-C object (in charge of sending commands to the railway amplifier) and every train present in the model, attaching them to the D\-C\-C.

It also registers all of the train commands in the intepreter.

At the end, sets the current\-\_\-train to the first one stored in the global trains array.
